CRUD Empleados (Full-Stack)

Aplicación para la gestión de personal conectada a múltiples backends, demostrando versatilidad en el ecosistema JavaScript.

Arquitectura del Proyecto

Este proyecto demuestra una implementación completa de una aplicación CRUD (Create, Read, Update, Delete) utilizando React para el frontend.

Lo que hace especial a este proyecto es su capacidad de interoperar con dos backends diferentes: una API diseñada en Laravel 9 y otra en Node.js (Express).

Stack Tecnológico

  • Frontend: React con Axios para consumo de servicios REST.
  • Backend 1: Laravel 9 (PHP) con base de datos relacional.
  • Backend 2: Node.js + Express para alta concurrencia.
  • UI: Bootstrap para estructura y SweetAlert2 para notificaciones premium.
