Crud Empleados (React)

Previous Next

Este es un crud donde podemos guardar datos básico de un empleado, consumiendo una api diseñada en laravel 9, donde tambien la desarrolle en nodejs usando el framework express. El frontend esta desarrollado en react usando axios para la comunicación con la api, y sus estilos usando bootstrapp, ademas de usar sweetalert2 y react-icons para los iconos.
