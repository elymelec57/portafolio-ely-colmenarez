Calculadora

Esta calculadora está realizada con el patron arquitectonico MVC, y orientado a objeto. El frontend está realizado con bootstrap 4 y el backend con php. En javascript usé las peticiones Ajax para conectarme al servidor, dichos datos están guardados en una base de datos Myqsl. Además todos los cálculos están realizados por javascript en el frontend.
